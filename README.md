# CyberAtaraxia-Training-BlueTeam
This is the first program of the CyberAtaraxia Training Suite, which is an extension of the CyberAtaraxia Suite designed for training. CyberAtaraxia Training - BlueTeam is an open source, free web-based app with the sole goal of guiding new commers in the Blue Teaming World into finding their way in this vast field.
